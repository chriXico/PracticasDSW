¿Que es HTML?

HTML son las siglas de (Hypertext Markup Languaje), que puede traducirse como lenguaje de marcas o marcado de hipertexto

El lenguaje HTML se emplea para crear las paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta de un navegador a otro y tambien puede cambiar de una veriona a otra de un mismo navegador pero suelen tener un nombre similar.

El lenguaje HTML sirve para describir la estructura basica de una pagina y organizar la forma en que se mostrara su contenido, ademas de que HTML permite incluir enlaces hacia otras paginas o documentos.

HTML se compone de etiquetas que se escriben entre los simbolos menor que y mayor que.

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creó la primera página web.
